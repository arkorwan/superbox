SuperBox
--------------------


-BOX.EXE	The executable program file
-BOX.PAS	The source code
-INSTALL.BAT	For installing program
-EGAVGA.BGI	For using graphic
-all files in CHR format	Fonts

In directory Data:
-all files in LVL format (A to L)	Levels
-all files in BGF format	Character image files
-HINTS.DAT	Hints data
-HIGHSC.DAT	High score data
-MINMOVE.DAT	Data for score calculation
